ExpressJs

To start practicing or "coding" in express. You would need to install Node in your system first. To download it click here.

Complete the installation by allowing it to do all the necessary steps.

I hope that you must be having a code editor with you. Open it (say VS Code) and create a file (say index.js) for its respect. To make your life easy with the "get" and "post" requests, install Thunder Client Plugin in the VS Code. It is similar as Postman. Hit Ctrl + Shift + ` to open a new terminal and throw the following codes. 

                
    $ npm init
    $ npm install express
    $ npm install -g nodemon
    $ npm install pug

Type your first lines of code in js (expressJs). It is expected to have the basic knowledge of NodeJs before entering ExpressJs.
